УСТАНОВКА 3X-UI НА СЕРВЕР

ГАЙД ПО УСТАНОВКЕ 3X-UI НА СЕРВЕР!
1. ОБНОВИМ ПАКЕТЫ УБУНТУ
apt update && apt upgrade -y
Теперь установим Curl — эта утилита нужна для загрузки файлов и пакетов в терминале. Сделаем это с помощью этой команды:
apt install curl -y
2. СКАЧАЕМ 3X-UI С ГИТХАБА
bash <(curl -Ls https://raw.githubusercontent.com/mhsanaei/3x-ui/master/install.sh)
после успешного скачивания
проверим работаспособность x-ui
и должна вылезти информация о аргументах x-ui
3. x-ui start
Смотрим Данные о Админке
пишем 10
и там должно вылезти
[INF] current panel settings as follows:
username: ваш логин
password: ваш пароль
port: ваш порт
webBasePath: путь к Админке
4. переходим в админку
https://ip_сервера:ваш порт/путь к Админке
5. настроика подключения
нажимаем на Добавить подключение
и все как тут
