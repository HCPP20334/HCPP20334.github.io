> 3X-UI_SERVER_INSTALLATION_

> Posted: 18.03.25_ | ID: 203341_

> 1. Update Ubuntu packages:_

apt update && apt upgrade -y

> 2. Install Curl:_

apt install curl -y

> 3. Download 3X-UI:_

bash <(curl -Ls https://raw.githubusercontent.com/mhsanaei/3x-ui/master/install.sh)

> 4. Start service:_

x-ui start

> View admin panel details with option 10, then access at:_

> https://your_server_ip:port/path_

> Connection setup:_

> Protocol: vless_
> Port: 443_
> Transmission protocol: TCP(RAW)_
> Security: REALITY_
> uTLS: chrome_
> Dest: www.apple.com:443_
> SNI: www.apple.com_

> Click "Get New Cert" to auto-generate keys._
